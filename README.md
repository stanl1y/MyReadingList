# 📚 MyReadingList — Interactive Paper Reading Guides

Bilingual (中文 / English), interactive HTML reading guides for CS / AI papers,
organized into research threads (a "grape bunch").

### 🔗 Live: **https://stanl1y.github.io/MyReadingList/**

Each guide turns a dense paper into a six-tab walkthrough — **Background · Problem ·
Method · Results · Critique · Takeaway** — with:

- 🎨 a 5-color highlight protocol (🔴 problem · 🔵 method · 🟢 result · 🟡 assumption · 🟣 limitation)
- 🌐 a 中 / EN language toggle (top-right, remembered across pages)
- ➗ real LaTeX math (KaTeX, bundled offline — no CDN)
- 🖼️ the paper's **real figures**, cropped from the PDF and embedded
- 🧵 a homepage that clusters related papers into **threads** with lineage arrows + a synthesis blurb

## How it's built

Generated by a custom `paper-reading-guide` workflow: a PDF tool fetches the paper,
extracts text, and renders + crops figures; the guide is written to a 6-tab bilingual
template; an index builder regenerates the clustered homepage from each guide's embedded
`paper-meta` tag (clusters & lineages are curated in the builder's config).

## Local viewing

```bash
cd papers && python3 -m http.server    # then open http://localhost:8000
```

Every guide is a single self-contained `.html` (offline-capable; math + figures included).
