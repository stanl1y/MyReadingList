# CS 論文 HTML 導讀 — 可重複使用 Prompt 模板

> 用法:在 Claude.ai(網頁版)開新對話 → 上傳論文 PDF → 把下面 `=== PROMPT 開始 ===`
> 到 `=== PROMPT 結束 ===` 之間整段貼上 → 右側 Artifacts 會即時渲染成 HTML 導讀頁。
> 想存檔就按 Artifact 右上角的下載,或在 Claude Code 裡叫我把它存成 .html。

---

=== PROMPT 開始 ===

你是一位資深的 AI研究者,專長是幫我把論文「導讀」成一頁排版精美、
可互動的 HTML。請仔細讀完我附上的論文,產出**單一檔案、可離線開啟**的
HTML(所有 CSS 與 JS 都 inline,不依賴任何外部資源)。

## 一、整體設計規範(務必照做)

**配色與質感**
- 頂部固定標題列:深藍底(#1a2b4a 附近)+ 金色標題字(襯線體)
- 內容用「紙張感」白色圓角卡片 + 柔和陰影,背景淺米色
- 標題用襯線字(serif),章節編號/標籤用等寬字(monospace),內文用無襯線

**頂部固定標題列**
- 左側標題:`CS PAPER READING GUIDE`
- 右側放一排「標色圖例(legend)」,共 5 種,對應下方標色協定

**標色協定(整份文件一致,這是核心)**
- 🔴 PROBLEM / GAP — 問題、現有方法的缺口、痛點
- 🔵 METHOD / IDEA — 核心方法、關鍵 insight、技術貢獻
- 🟢 RESULT / EVIDENCE — 實驗數據、量化結果、勝過 baseline 的證據
- 🟡 ASSUMPTION / CLAIM — 隱含假設、作者的主張、適用前提
- 🟣 LIMITATION / FUTURE — 限制、失敗案例、未來方向

正文中每個關鍵詞句,用對應顏色的底線或 highlight 標記。

**分頁導覽列(tab nav,用 JS 切換顯示)**
編號 + 等寬字,共 6 頁:
`01 BACKGROUND` · `02 PROBLEM` · `03 METHOD` · `04 RESULTS` · `05 CRITIQUE` · `06 TAKEAWAY`

## 二、各分頁內容

**01 BACKGROUND(背景動機)**
- 頂部資訊卡:標題、作者、年份、會議/期刊、一句話 TL;DR(繁中)
- 「如果只有 30 秒」:3 個 bullet 把整篇壓縮成核心貢獻
- 領域背景敘事(左欄長文 + 關鍵句標色),右欄旁註卡解釋專有名詞

**02 PROBLEM(問題診斷)**
- 用「現況 vs 痛點」對比卡,點出這篇要解決什麼、為何重要
- 「Gap 表」:現有方法 | 各自的限制 | 本文如何補上(三欄)

**03 METHOD(方法核心)**
- 用文字流程(或 HTML/CSS 畫的方塊箭頭圖)說明 pipeline
- 標出最關鍵的 1~2 個 idea,各配一句白話解釋(類比也可)
- 關鍵公式/符號:列出來並用白話說明每個符號代表什麼

**04 RESULTS(實驗證據)**
- 「主結果表」:方法 × 資料集/指標,本文數字 vs baseline,勝出處標綠
- 「Ablation 解讀」:每個被拿掉的元件 → 掉了多少分 → 代表它多重要
- 一句話總結:實驗到底證明了什麼、沒證明什麼

**05 CRITIQUE(批判提問)**
- ⚠️ COMMON PITFALL 卡:讀這篇最容易誤解或高估的地方
- 可被質疑的點:實驗設定、baseline 公平性、是否 cherry-pick
- 給審稿人/讀書會的 3~5 個犀利問題(`Q1 Q2 Q3`...)

**06 TAKEAWAY(帶走重點)**
- 對我研究的啟發:可延伸方向、可借用的技巧
- 值得引用的點(配一句該怎麼引用的情境)
- 一句話結論

## 三、互動與細節
- 分頁可點擊切換;標色詞 hover 時顯示該類別名稱
- 雙語:分析以繁體中文為主,專有名詞與關鍵術語保留英文原文
- 所有「事實/數字」務必來自論文本身;若是你的推論或補充,明確標註
  「[導讀補充]」,不要與論文原意混淆

請直接產出完整 HTML。

=== PROMPT 結束 ===

## 進階變體(視需要加在 prompt 結尾)

- **多篇橫向比較**:讀完同領域 N 篇後 →「請生一張橫向比較表:各方法的
  假設 / 資料集 / 核心技巧 / 主要結果 / 限制,並用一段話總結這條研究線的演進」
- **語音導讀**:在 HTML 右下角加一個播放鈕,用瀏覽器內建 Web Speech API
  朗讀 TL;DR 與每頁重點(免外部資源)
- **個人論文庫**:在 Claude Code 裡叫我把每篇存成 `papers/作者_年份.html`,
  再生一個 `index.html` 當可搜尋目錄
